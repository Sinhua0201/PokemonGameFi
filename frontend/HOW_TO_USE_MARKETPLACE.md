# 如何使用 Marketplace (OCT 版本)

## 💎 支付方式

现在 Marketplace 使用 **OCT (OneChain 原生代币)** 进行支付，不再需要 POKEMON 自定义代币。

## 🛒 购买 NFT

### 步骤 1: 获取 OCT
从 OneChain 水龙头获取 OCT：
```bash
curl -X POST https://faucet.onechain.ai/gas \
  -H "Content-Type: application/json" \
  -d '{"FixedAmountRequest":{"recipient":"YOUR_ADDRESS"}}'
```

或访问: https://faucet.onechain.ai

### 步骤 2: 浏览 Marketplace
1. 访问 Marketplace 页面
2. 浏览可用的 Pokemon 和 Egg NFTs
3. 使用筛选器查找你想要的 NFT

### 步骤 3: 购买
1. 点击你想购买的 NFT
2. 查看详细信息和价格（以 OCT 显示）
3. 点击 "Buy" 按钮
4. 确认交易
5. 等待交易完成

## 📝 上架 NFT

### 步骤 1: 选择要上架的 NFT
1. 在 Marketplace 页面点击 "List Your NFTs" 标签
2. 选择你想上架的 Pokemon 或 Egg

### 步骤 2: 设置价格
1. 输入价格（以 OCT 为单位）
2. 系统会显示你将收到的金额（扣除 2.5% 手续费）

### 步骤 3: 确认上架
1. 点击 "List NFT" 按钮
2. 确认交易
3. 等待交易完成

## 💰 价格说明

- 所有价格以 **OCT** 显示
- Marketplace 收取 **2.5%** 手续费
- 例如：如果你以 1 OCT 上架，你将收到 0.975 OCT

## ❌ 取消上架

1. 在 "My Listings" 标签查看你的上架
2. 点击 "Cancel" 按钮
3. 确认交易
4. NFT 将返回到你的钱包

## 🔍 常见问题

### Q: 我需要 POKEMON 代币吗？
**A:** 不需要！现在只需要 OCT。

### Q: 如何获取 OCT？
**A:** 从 OneChain 水龙头获取，或者通过出售 NFT 赚取。

### Q: 手续费是多少？
**A:** 2.5%，从卖家收取。

### Q: 购买失败怎么办？
**A:** 确保你有足够的 OCT，并且 NFT 还在上架中。

### Q: 可以用其他代币支付吗？
**A:** 目前只支持 OCT。

## 🎯 优势

相比之前的 POKEMON 代币版本：

✅ **更简单** - 不需要额外获取自定义代币
✅ **更直观** - 直接使用钱包中的 OCT
✅ **更快速** - 减少了代币转换步骤
✅ **更低 Gas** - 交易更简单，Gas 费更低

## 📞 需要帮助？

如果遇到问题：
1. 检查你的 OCT 余额
2. 确认 NFT 还在上架中
3. 查看浏览器控制台的错误信息
4. 联系支持团队
